## Internals
Performing deeper enumeration. A lot of it is repeat from previous enumeration modules. Some common actions in Linux:

- Network interfaces: `ip a`, `cat /etc/hosts`
- `lastlog`, how frequent, when last login
- `w` current logons
- `history` 
- `find / -type f \( -name *_hist -o -name *_history \) -exec ls -l {} \; 2>/dev/null` find special history files.
- `ls -la /etc/cron.daily` cron jobs
- proc / procfs file system, containing info about system processes, hardware, system information. Can be used to view and modify kernel settings. Virtual and does not exist as real filesystem, but dynamically generated by kernel. Can set certain parameters such as process priority, scheduling, memory allocation
```shell-session
find /proc -name cmdline -exec cat {} \; 2>/dev/null | tr " " "\n"
```

List of installed packages
```shell-session
apt list --installed | tr "/" " " | cut -d" " -f1,3 | sed 's/[0-9]://g' | tee -a installed_pkgs.list
```
Don't forget sudo version for  exploits:
```shell-session
sudo -V
```
Binaries
```shell-session
ls -l /bin /usr/bin/ /usr/sbin/
```
https://gtfobins.github.io/
Platform for list of binaries to exploit. Compare existing binaries with GTFObins
```shell-session
 for i in $(curl -s https://gtfobins.github.io/ | html2text | cut -d" " -f1 | sed '/^[[:space:]]*$/d');do if grep -q "$i" installed_pkgs.list;then echo "Check GTFO for: $i";fi;done
```

syscall and signal processing. Can monitor security related activities and identify potential attack vectors.

```shell-session
strace ping -c1 10.129.112.20
```
more configuration
```shell-session
find / -type f \( -name *.conf -o -name *.config \) -exec ls -l {} \; 2>/dev/null
```
Dont forget scripts
```shell-session
find / -type f -name "*.sh" 2>/dev/null | grep -v "src\|snap\|share"
```
running services by user
