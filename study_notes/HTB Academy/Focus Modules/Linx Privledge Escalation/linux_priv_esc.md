# Linux Privilege Escalation


During a more red team focused exercise, I may be given low privileged shell on Linux host and need to perform privilege escalation to root.

Allows for capture traffic, sensitive files, and if domain joined (device connected to centralized network managed by AD controller, so shared resources, centralized security, admin policy), gain NTLM hash (user password) and enumerating and attacking active directory.
