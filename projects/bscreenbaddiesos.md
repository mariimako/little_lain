# Overview

Creation of operating system with a team of four for OS Design and Implementation course. Microkernel architecture, based upon the Barrelfish architecture.

We were able to implement a physical memory manager, virtual memory and paging, a multi-threading, IPC, UMP. 

OS development gave me an appreciation of the importance of very long design processes and planning. Weeks of planning allowed us to properly implement efficeint and original elements into our final project. Our benchmark tests proved we had similar efficiencies to non-microkernal operating systems regarding allocation times.
